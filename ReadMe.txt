Compile the client.c and server.c files.
FOR SERVER
	$ gcc server.c -o server1
	
FOR CLIENT 
	$ gcc client.c -o client1

Now run the server1 file
	$./server1

Now run the client1 file
	$./client1

1. First register or sign up as an admin.

2. Now sign in into admin account and add trains/ modify trains.

3. Now you can create customer account or agent accounts using appropriate sign up.

4. You can now sign in using customer or agent account to book tickets and perform other operations.

5. You can view the bookings and update or cancel the bookings.
